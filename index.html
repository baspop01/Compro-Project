Computer Programming Project

-. เครื่องตรวจจับควัน MQ-2 .-

เข้าสู่เนื้อหา


PURPOSE

เพื่อทำการทดลองและอธิบายการทำงานของวงจรเครื่องตรวจจับควัน MQ-2 ซึ่งเอาไว้ตรวจจับของกลุ่มควันและตรวจจับแก๊สได้อีกหลากหลายชนิด และสามารถนำไปประยุกต์ใช้ในการตรวจจับกลุ่มควันที่เกิดจากการเผาไหม้สิ่งต่างๆ ที่สามารถก่อให้เกิดอันตรายต่อสิ่งมีชีวิตและอื่นๆที่ทำให้เกิดความเดือดร้อนได้

What can be used for???

Fire Alarm

แจ้งเตือนเมื่อสามารถจับกลุ่มควันในระดับความหนาของควัณที่เกิดอันตรายได้

Gas Leak

ป้องกันการรั่วไหลของถึงแก๊สที่บรรจุแก๊สชนิดจำพวก LPG Methane ที่สามารถก่อให้เกิดประกายไฟได้

Other

และสิ่งอื่นๆอีกมากมายที่สำคัญต่อการยับยั้งสิ่งที่ก่อให้เกิดอันตรายกับชีวิตและทรัพสินย์ เช่น การวัดระดับปริมาณแอลกอฮอล์ในร่างกาย เป็นต้น

Equitment

  • Arduino Uno R3
  • Sensor MQ-2
  • หลอดไฟ LED สีเขียว แดง
  • ตัวต้านทาน ค่าความต้านทานที่ 220 Ohm
  • Breadboard
  • ไฟแช็ก (เอาไว้สำหรับทดลอง)

    • ตัวอย่างและการทำงานของวงจร

    เมื่อมีกลุ่มควัน ก๊าซ แก๊ส มาโดนบริเวณ Sensor MQ2 จะมีการส่งสัญญาณ analog ไปยัง Arduino และเมื่อความหนาแน่นของ ก๊าซ แก๊ส ควัน เข้าสู่ sensor ก็จะทำให้ค่า analog สูงขึ้นแล้วส่งเก็บไว้ไปยัง Arduino หากค่าที่รับมีค่าต่ำกว่าที่ตั้งไว้ (ตัวอย่างมีค่าความหนาแน่นที่ 400) ไฟสีเขียวจะสว่างเตือนว่าปลอดภัยยังไม่เกินค่าที่ตั้งไว้ และในทางกลับกันหากมีกลุ่มควัน แก๊ส ก๊าซ มีความหนาขึ้นสูงกว่า ค่าที่ตั้งไว้ ไฟสีแดงจะสว่างเพื่อเตือนว่าแก๊สหรือกลุ่มควันที่ตรวจได้นั้นอันตรายแล้วนะ
